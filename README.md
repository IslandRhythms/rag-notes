# rag-notes

## Running this app

1. `npm install`
2. Copy `.env.example` into `.env` and provide your Astra connection details and OpenAI key
3. Run `npm run seed`
4. Run `npm start`
5. Visit `http://localhost:3000` and you should see the following screen
